    DOWN    :TO MAKE A HEIGH KICK

    F4      :FULL SCREEN / WINDOWED
    ESC     :EXIT

    OBJECTIVE
    DESTROY EVERYTHING
    DESTROY THE BALLS TO MAKE YOUR WAY
